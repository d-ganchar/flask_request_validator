Key features
------------
- Easy and beautiful
- Type conversion
- Extensible
- GET, FORM, nested JSON, headers and files validation
- Post validation hooks
- Custom error messages
- Supports [Flask-RESTful](https://flask-restful.readthedocs.io/en/latest/)

#### How to install:

```
$ pip install flask_request_validator
```
